About React Native

  • Javascript framework for creating native applications
  • released in 2015 by Facebook
  • open source!
  • allows to create iOS, Android and UWP applications

So what's the difference?

  • uses native component

Pros:

  • smooth interface
  • most js library works

Cons:

  • no CSS, nor HTML
